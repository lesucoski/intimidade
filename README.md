# Intimidade - Wellness Experience 🖤
*Luxo, Conexão e Descoberta para Casais.*

Este é o repositório oficial da marca **Intimidade**. Um ecossistema digital desenhado para transformar a rotina em rituais inesquecíveis.

## 🚀 Funcionalidades
- **Boutique Exclusiva:** Vitrine de alta conversão com curadoria de produtos premium.
- **Atelier da Intimidade:** Quiz interativo para montagem de kits personalizados.
- **Oráculo da Intimidade:** Jogo de dados interativo para guiar a noite do casal.
- **Atmosfera Sensorial:** Hub de áudio e tutoriais educacionais (Nó de Seda).

## 🛠️ Tecnologias
- HTML5 / CSS3 (Tailwind CSS 4.0)
- JavaScript (Vanilha)
- Design System: Dark Luxury Aesthetic

## 📦 Como rodar localmente
1. Clone o repositório.
2. Abra o arquivo `index.html` em seu navegador.
